## Les Adjectifs aprés le nom

Following are the categories of adjectives that are always placed after the noun:
1. Les formes. ( [[C'est une porte carre]].  This is a square door )
2.  Les nationalités. ([[C'est un garton indién]] - This is an Indian boy )
3. Les couleurs. ([[C'est une maison rouge]] - This is a red house )
4. La religion. ([[C'est une fille hindoue]] - This is a Hindu girl )
