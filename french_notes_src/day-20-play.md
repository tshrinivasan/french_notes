# Jouer ( pour les instruments )

Par example :

- [[ je joue du piano ]] ( jouer + du )

- [[ il joue de la guitare ]] ( jouer + de la )

- [[ les enfants jounet de l'harmonica ]] ( jouer + de l' )

- [[ mes amis jounet des marcas ]] ( jouer + des )
