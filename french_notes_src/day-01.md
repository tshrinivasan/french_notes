## Silent Rule
most last letters are silent

 [[ bouffet ]] - last t is silent here


##  when é is there, spell as aee.

- [[ café ]] ( pronounce last letter) - coffee and coffee shop

- [[ Résumé ]] - Reasumea ( read like this )

- [[ chauffeur ]] - private driver => shofer

cha => sha

- [[ professeur ]] => professeur ( same sound )

## change sound

- ch => sh
- au => o
- eur => ar

- [[ Bon Appétit ]]  => Bo apethee ~ enjoy your food.

## Soft letter rule
t,d,g have soft pronouncations.

- t => th
- d => dh
- g => jh

## silent last letter

- [[ Faux ]] => foo ( ~ artificial, fake )
- [[ Faux Leather ]] => foo leather


## exception 1

if word ends with vowel, a,i,o,u spell it. ( except e )

- [[ stylo ]] => sthlo  - pen
- [[ Adeiu ]] => adeeiu


## exception 2

if e is é, then it should be spelled as ea.


## exceptopn 3

if a word ends with c, pronounce it.

- [[ chic ]] => shek  ~ fashionable

- [[ sac ]] => sac ~ handbag

## pronounce these

c,l,r,f = if word ends with these, pronounce it.
