day 4 - dialogs

  • a tout a l’heure => athu thalar =

  • Monsieur (M) => Ma sier ~ Sir, Mr

  • Madame (Mme) => madham ~ Ma’am

  • Mademoiselle (Mlle) => Mad muwasel ~ Miss

  • D’acccord ( dac ) => Dacor~ Okey

  • Entendu => anthan duu ~ Got it

  • S’il vous plait => sil vou plat ~ please ( forma;l)

  • sil te plait => sil the plat ( te) informal ~ please

  • oui => wui ~ yes

  • Non => no

  • Je M’appelle => jemapple (name) ~ my name is

  • Merci => mersi ~ thank you

  • De rien=> Dha ria ~ No biggie

  • Je vou

  • desole =>Desolee ~ sorry ( single e = male, double ee = female )

  • Enchante => Enchantee => onshanthe ~ nice to meet you

  • pas de probleme ~ pa dha problem ~ no problem

  • et toi? ~ and you?

  • Bien ~ good

  • Oui => wee ~ yes
