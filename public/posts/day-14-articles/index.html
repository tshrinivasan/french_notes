Day 14 - Les Articles

Les Articles

Les Articles

A, an, the

I am a boy.
This is a mobile phone.
This is the mobile phone of Hawa.

Il y a deux type d’articles :

Les articles indéfinis (a, an) Les articles définis (the)

Les Articles Indéfinis

French Articles and Examples

Un (nom masculin)

(a, an)

  • Un livre — a book
  • Un garçon — a boy
  • Un restaurant
  • Un papier
  • Un cheval
  • Un rideau — a curtain
  • Un professeur
  • Un crochet
  • Un parlement
  • Un mur
  • Un journal — a newspaper

Une (nom féminin)

(a, an)

  • Une porte — a door
  • Une princesse
  • Une célébration
  • Une fille
  • Une télévision
  • Une police
  • Une parade
  • Une gamme
  • Une leçon
  • Une épice

Des (nom pluriel)

(some)

  • Des livres — some books
  • Des portes — some doors
  • Des garçons
  • Des rideaux

*(Words ending in eau, eu, au form plural with x)

  • Des filles — some girls
  • Des chevaux — some horses
  • Des journaux — some newspapers

C’est un livre. = This is a book.
C’est une porte. = This is a door.
C’est un rideau. = This is a curtain.

Ce sont des livres. = These are some books.
Ce sont des portes.
Ce sont des rideaux.

Les Articles Définis (the)

Structure:
Sujet + verbe + article + objet

  • le (masculin)
  • la (féminin)
  • l’ (devant une voyelle ou un h muet)
  • les (pluriel)

Exemples avec le

  • Le livre — the book
  • Le mur
  • Le tableau
  • Le chauffeur
  • Le gouvernement
  • Le bureau
  • Le journal
  • Le drapeau
  • Le carnaval
  • Le diagramme
  • Le chat

Exemples avec la

  • La fenêtre — the window
  • La rage — the anger
  • La gomme
  • La fleur
  • La pluie — the rain
  • La page
  • La prédiction
  • La parade
  • La messe
  • La sueur
  • La justice

Exemples avec l’

  • L’horloge — the wall-clock
  • L’été
  • L’affaire
  • L’épice
  • L’érable
  • L’année
  • L’exercice
  • L’étudiant
  • L’avion
  • L’éléphant

Exemples avec les

  • Les livres
  • Les fenêtres
  • Les horloges
  • Les tableaux
  • Les parades
  • Les exercices
