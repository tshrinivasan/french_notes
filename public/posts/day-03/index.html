Day 03 - Brands & Greetings

day 3

  • Les Marques => Le Marque ~ the brands

  • Hermes => ermes

  • Chanel => shanel

  • Kenzo => Kenzo

  • Louis Vuitton => lui vithon

  • Cartier => Cartiea ( r silent )

  • Jean Paul Guatier => Zha Poul Gothiae (n, r silent )

  • Yues Saint Laurent => Ev san Loran

  • Dior => Dhior

  • L’Oreal => Loreal

  • Celine => selin

  • Christian Louboutin => christhia Lubuta ( in => aaa , at the end only)

  • Lancome => loncom

  • Balmain Paris => balmaa pari ( in=> aa)

  • Mont Blanc => mon blan

  • long champs => Lo shaun

  • mps => n ( rule )

  • chole => cloie

  • Roger Dubuis => Rosher diu-biu-i

Note

the below three are with same sound.

  • Lanvin => Laavaa

  • Vin => Vaa ~ wine

  • Vingt => Vaa ~ 20 ( number) ( g,t silent )

nasal sounds

ean => aaon ( say it with one nose closed) en an

greetings

  • Les Salutations => Le sa-liu-tha-sio

  • Salut => saa-liu ~ hey,hi ( informal way )

  • Bonjour => bonjur ~ hello ( formal)

  • BonSoir => bon swar ~ good evening

  • Bonne nuit => Bon new ii ~ good night

  • Ça va ? => savaa ? ~ all good? ( spoken, informal)

  • Ça va=> sava ~ yes. all good

  • Comment allez-vous? ~ how are you? ( vous = formal ) => komo thaaley vu

  • comment vas-tu ( tu = informal ) => komo vaa thiuu ~ how are you ?

  • tu vas bien ~ how are you

  • Je vais bien => zhe ve bian ~ I am fine

  • Bonne journée => bonn zhurne ~good day

  • Bon voyage => bon voyaj ~ have a good trip

  • Bonne soirée ~ good eveing/enjoy your evening => bon swarey

  • a bientot => a bieatho ~ see you soon

  • a demain => a dhama ~ see you tomorrow

  • Au revoir => Ou ravva ~ bye ( formal )

  • Tchao => tchao ~ bye ( informal) ( ch is ch here, when followed by t)
